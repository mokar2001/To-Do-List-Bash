#### Execution Permisson
`sudo chmod +x todo.sh`

#### Add Task
`./todo.sh add -t <task_name> [-p L|M|H]` 

#### Find Task
`./todo.sh find <task_name_contains>`

#### Done Task
- `taask_id` can be found with `find <task_name_contains>`

`./todo.sh done <task_id>`



#### Show Tasks
`./todo.sh list`

#### Clear Tasks
`./todo.sh clear`
